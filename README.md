# 🚀 Centro de Mandos (Command Center)

**La navaja suiza para automatizar flujos de trabajo de desarrollo.**

[**🌐 Ver Demo en Vivo**](https://commandcenterdev.netlify.app)

¿Cansado de pensar nombres de ramas consistentes? ¿Olvidaste el comando exacto para limpiar la caché de Laravel dentro de Docker? El **Centro de Mandos** es una herramienta web premium diseñada para transformar ideas y tareas en comandos listos para ejecutar.

---

## 🛠️ ¿Qué puedes hacer?

### 🌿 Git Flow Maestro
Transforma el título de una tarea de Jira o Trello en una estructura de Git profesional.
- **Smart Detection:** Escribe "Fix error en login" y el sistema detectará automáticamente que es un `fix/`.
- **Generación de Commits:** Crea mensajes de commit estandarizados basados en tu tarea.
- **Secuencia de Comandos:** No solo te da la rama, te da el flujo completo: `checkout`, `pull`, `commit` y `push`.
- **Modo Terminal:** Visualiza tus comandos como si estuvieras en una consola real.

### 📦 Node.js Quick-Actions
Gestiona tus dependencias sin memorizar flags.
- Cambia entre `NPM`, `Yarn` o `PNPM` al instante.
- Acciones rápidas para limpieza de `node_modules`, auditorías de seguridad e inicialización de TypeScript.

### 🐳 Docker & Laravel Vault
Una biblioteca inteligente de comandos complejos para contenedores.
- **Plantillas Dinámicas:** Los comandos se adaptan a tu archivo `docker-compose.yml` y al nombre de tu servicio (ej. `api`).
- **Laravel Ready:** Limpieza de caché, migraciones y ejecución de Tinker dentro del contenedor con un solo clic.
- **Power Actions:** Reconstrucción completa del entorno y depuración de variables de entorno (MAIL, DB, etc.).

## ✨ Características Premium

- **Diseño Glassmorphism:** Interfaz oscura, minimalista y elegante optimizada para sesiones nocturnas de código.
- **Bóveda de Comandos (Vault):** Un buscador integrado para encontrar ese comando de Git o Docker que siempre olvidas.
- **Historial Reciente:** Acceso rápido a tus últimas ramas generadas para que no pierdas el hilo de tus tareas.
- **Multi-idioma:** Soporte completo para Inglés y Español.
- **Validación en tiempo real:** Avisos visuales si el nombre de tu rama es demasiado largo o contiene caracteres prohibidos.

---

## 💡 Cómo se usa

1.  **Selecciona tu módulo:** Git, Node o Docker desde la barra de navegación superior.
2.  **Escribe tu tarea:** Introduce el nombre de lo que estás haciendo (ej: "Añadir notificaciones toast").
3.  **Refina:** Elige el tipo (Feature, Fix...) y el sistema generará todo por ti.
4.  **Copia y Ejecuta:** Haz clic en los comandos o en el botón "Copiar Todo" y pégalos en tu terminal. **¡Listo!**
